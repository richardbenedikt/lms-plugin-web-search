# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-02-15
### Added
- Add mandatory citation instructions to `visit_website` output. [@Richard B.](https://github.com/richardbenedikt)
- Inject current date into `web_search` tool to support temporal queries. [@Richard B.](https://github.com/richardbenedikt)
### Changed
- Update `visit_website` tool to use `jsdom` and `@mozilla/readability` for robust content parsing instead of regex. [@Richard B.](https://github.com/richardbenedikt)

## [1.0.0] - 2026-02-14
Initial release. [@Richard B.](https://github.com/richardbenedikt)
